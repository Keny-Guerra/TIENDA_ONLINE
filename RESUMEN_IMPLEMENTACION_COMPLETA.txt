🚀 CÓMO USAR

   2. npm install (si es primera vez)
   3. npm start
   4. Esperar: "Servidor ejecutándose en puerto 3000"
   5. Abrir: http://localhost:3000

   5. Completar pago
   6. Ver pedido en "Mis Pedidos"
   7. Ver estadísticas en "Reportes"
